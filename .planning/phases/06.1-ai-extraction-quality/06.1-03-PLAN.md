  <action>
For the earthmoving schema, make these changes:

**Fields to keep at aiExtractable: false (do not enable):**
- hourmeter (sfOrder 6) — intentionally false; earthmoving hour meters not reliably readable
- odometer (sfOrder 9) — not applicable to most earthmoving equipment
- engine_number (sfOrder 12) — never infer engine numbers
- speeds, torque_rpm, boom_length, steering, final_drive, cabin, operator_station, attachments, config, screen_size, rims, pto, remotes, extras — too specific, no reliable inference

**Fields to flip to aiExtractable: true AND add aiHint:**
- engine_model (sfOrder 11): `aiHint: 'Infer from make/model/year (e.g. Cat C7.1, Komatsu SAA6D107, Cummins QSB6.7). Model-specific constant.'`
- horsepower (sfOrder 13): `aiHint: 'Infer from make/model/year. Digits only, no units (e.g. 168 for Cat 320D). Known spec for common machines.'`
- gross_hp (sfOrder 14): `aiHint: 'Infer from make/model/year. Gross HP is slightly higher than net HP. Digits only.'`
- fuel_type (sfOrder 15): `aiHint: 'Earthmoving equipment is almost always Diesel. Only return non-Diesel if clearly evidenced (e.g. electric excavator badge).'`
- emissions_tier (sfOrder 16): `aiHint: 'Infer from year: Tier 4 Final (post-2014), Tier 4 Interim (2011–2014), Tier 3 (2006–2011). Australia may be 1 year behind US.'`
- drive_type (sfOrder 17): `aiHint: 'Infer from subtype and model: Crawlers are Track Drive, wheel loaders/graders are 4WD. e.g. Track Drive, 4WD.'`
- transmission (sfOrder 18): `aiHint: 'Infer from make/model (e.g. Cat automatic planetary, Komatsu PCSS, ZF powershift). Common: Automatic, Powershift, Hydrostatic.'`
- tare (sfOrder 29): `aiHint: 'Operating weight from machine specs. Digits only in kg. Well-documented for common models (e.g. Cat 320D ≈ 21000 kg).'`

**Fields already aiExtractable: true — add aiHint:**
- pin (sfOrder 1): `aiHint: 'Build plate: Product Identification Number (PIN). Alphanumeric, typically 17 chars for Cat. Never infer — only extract if visible.'`
- serial (sfOrder 2): `aiHint: 'Build plate: Serial Number. Format varies by manufacturer. Never infer — only extract if directly visible.'`
- vin (sfOrder 3): `aiHint: 'Build plate or chassis: 17-character VIN if present. Many earthmoving machines use PIN instead of VIN. Never infer.'`
- make (sfOrder 5): `aiHint: 'Build plate: manufacturer name (e.g. Caterpillar, Komatsu, Hitachi, Volvo, Liebherr, John Deere, CASE).'`
- model (sfOrder 7): `aiHint: 'Build plate: model designation (e.g. 320D, PC200, ZX200, EC220, LiG952H). Read exactly as printed.'`
- year (sfOrder 8): `aiHint: 'Build plate: Year of Manufacture or DOM. 4-digit year only. May be deduced from serial number prefix for some manufacturers.'`
- engine_manufacturer (sfOrder 10): `aiHint: 'Build plate or engine badge. Infer from make/model (e.g. Cat→Caterpillar, Komatsu→Komatsu/SAA, Volvo→Volvo D-series).'`
  </action>
