## Spacing Scale

Standard 8-point scale in use throughout the codebase. No new exceptions for this phase.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (icon `gap-2` = 8px; icon-only `gap-1` = 4px) |
| sm | 8px | Compact element spacing, badge padding |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, page-level inner padding |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Asset type selector buttons: `min-h-[80px]` — maintains touch target compliance (matches existing `AssetTypeSelector` pattern)

- Textarea minimum height: `min-h-[80px]` — existing pattern for `damage_notes` and `extras` fields

## Color

Slattery Auctions brand — dark green dominant, navy accent. Dark mode only; no light mode.

| Role | Value | CSS Variable | Usage |
|------|-------|-------------|-------|
| Dominant (60%) | #166534 (oklch 0.38 0.1 148) | `--background` | Page background, body surface |
| Secondary (30%) | #14532D (oklch 0.34 0.1 148) | `--card` / `--secondary` / `--muted` | Cards, sidebar, field inputs (`bg-white/5` overlay), field row dividers |
| Accent (10%) | #1E3A5F (oklch 0.29 0.07 248) | `--primary` / `--accent` / `--border` | See reserved list below |
| Destructive | #F87171 (oklch 0.74 0.15 20) | `--destructive` | Destructive actions only |

**Accent (#1E3A5F navy) is reserved for:**
- Selected asset type button border (`border-white` overrides to white when selected — unselected uses `border-[#1E3A5F]`)
- Unselected asset type tile border
- Input focus ring (`focus:ring-[oklch(0.29_0.07_248)]`)
- Form field borders (`border-white/15`)
- Low-confidence field left border accent: `border-l-red-500/40` (red-500 at 40% opacity, not the navy accent)
- Medium-confidence field left border accent: `border-l-amber-400/40`

**Supporting overlay colors (not primary palette — applied via opacity):**
- `bg-white/5` — input field backgrounds
- `bg-white/10` — selected state overlay on asset type buttons
- `text-white/65` — muted label text (maps to `--muted-foreground`)
- `text-white/30` — placeholder text in inputs

## Interaction Contracts

### Asset Type Selector tile (marine addition)
- **Default state:** `border-[#1E3A5F] bg-[#14532D]` — matches all other type tiles
- **Hover state:** `hover:bg-white/5` — matches existing tiles
- **Selected state:** `border-white bg-white/10` — matches existing tiles
- **Icon:** `Anchor` from lucide-react, `w-8 h-8 text-white`
- **Label:** "Marine" — `text-sm text-white text-center leading-tight`
- **Layout:** `flex flex-col items-center justify-center gap-2 rounded-lg p-4 min-h-[80px]`

### Motor Type select field
- Renders via existing `FieldRow` select branch
- Options: `['Inboard', 'Outboard', 'Stern Drive', 'Jet Drive', 'Electric']`
- Placeholder: "Select…" — matches existing select pattern
- `defaultValue` prop for pre-fill restoration — matches Phase 11 pattern

### Inspection priority inputs (pre-extraction structured inputs)
Three fields displayed on photos page before AI extraction runs:
1. **HIN** — label "HIN", inputMode="text", placeholder consistent with VIN pattern
2. **Engine Hours** — label "Engine Hours", inputMode="numeric"
3. **LOA** — label "LOA", inputMode="text"

Renders through existing `InspectionNotesSection` — no new component needed.

### Confidence badge rendering
- `high` confidence: no left border highlight (field value trusted)
- `medium` confidence: `border-l-2 border-l-amber-400/40 pl-3`
- `low` / `not_found` confidence: `border-l-2 border-l-red-500/40 pl-3`
- Applies to all 14 AI-extractable marine fields
