## Scope Note

Phase 16 is a schema-only change. The `AssetSubtypeSelector` component and the 3-step new-asset wizard already exist and handle all visual output. The UI change is:

1. Agriculture, Forklift, and Caravan now have populated `subtypes` arrays — the wizard Step 3 (`AssetSubtypeSelector`) renders automatically for these types with no component changes.
2. Existing Truck, Trailer, Earthmoving, Marine, and General Goods subtype selectors render updated lists from their replaced arrays.

No new components are introduced. No layout changes. No navigation changes.

## Spacing Scale

Standard 8-point scale — matches established project patterns in `new/page.tsx` and `AssetSubtypeSelector.tsx`.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, tight inline spacing |
| sm | 8px | Gap between subtype buttons (`gap-2`) |
| md | 16px | Horizontal page padding (`px-4`), subtype button horizontal padding (`px-4`) |
| lg | 24px | Section margin below header (`mb-6`) |
| xl | 32px | Section margin below content (`mb-8`) |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level bottom clearance |

Exceptions:
- Subtype button vertical padding: 12px (`py-3`) — not a standard token but matches existing component exactly; do not change.
- Minimum touch target on subtype buttons: 48px (`min-h-[48px]`) — preserves accessibility on mobile; do not change.
- Action button height: 44px (`h-11`) — matches existing wizard CTA; do not change.

- Step 3 (subtype) renders when `assetType` is set — already handles all 8 types including Agriculture, Forklift, Caravan once their `subtypes` arrays are populated.
