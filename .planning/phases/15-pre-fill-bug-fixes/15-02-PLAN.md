<objective>
Fix the unmount flush in `InspectionNotesSection` so that edits typed within 500ms of navigation are not silently lost on iOS back-button navigation.

Purpose: The current `useEffect` cleanup calls `saveInspectionNotes` (a Server Action), which returns a Promise. On iOS back-button navigation, the browser tears down the page before the Promise resolves, discarding the write. `navigator.sendBeacon` is guaranteed to deliver even after the page is torn down.

Output: New `POST /api/inspection-notes` route (auth + DB update), updated `useEffect` cleanup using `sendBeacon`, new route tests, updated unmount test.
</objective>

```typescript
function buildCombinedNotes(): string {
  const structuredLines = Object.entries(structuredValuesRef.current)
    .filter(([, v]) => v.trim())
    .map(([k, v]) => `${k}: ${v}`)
  return [
    ...structuredLines,
    notesRef.current.trim() ? `Notes: ${notesRef.current}` : '',
  ]
    .filter(Boolean)
    .join('\n')
}

// persistNotes delegates to buildCombinedNotes (no logic duplication):
const persistNotes = useCallback(() => {
  saveInspectionNotes(assetId, buildCombinedNotes())
}, [assetId])

// Unmount flush uses sendBeacon (guaranteed delivery after page teardown):
useEffect(() => {
  return () => {
    if (debounceRef.current) clearTimeout(debounceRef.current)
    navigator.sendBeacon(
      '/api/inspection-notes',
      new Blob(
        [JSON.stringify({ assetId, notes: buildCombinedNotes() })],
        { type: 'application/json' }
      )
    )
  }
}, [assetId])   // assetId is stable; buildCombinedNotes reads refs (not deps)
```

sendBeacon + Blob pattern (from research):
- `new Blob([jsonString], { type: 'application/json' })` sets Content-Type header correctly
- Same-origin request — no CORS preflight triggered
- Route handler can use `req.json()` to parse (standard pattern)
- Browser guarantees delivery even after page teardown

    ```typescript
    import { NextRequest } from 'next/server'
    import { createClient } from '@/lib/supabase/server'

    export async function POST(req: NextRequest) {
      const supabase = await createClient()

      // 1. Auth check
      const { data: { user } } = await supabase.auth.getUser()
      if (!user) return Response.json({ error: 'Unauthorized' }, { status: 401 })

      // 2. Parse request body
      let assetId: string, notes: string
      try {
        const body = await req.json()
        assetId = body.assetId
        notes = body.notes ?? ''
        if (!assetId) return Response.json({ error: 'assetId required' }, { status: 400 })
      } catch {
        return Response.json({ error: 'Invalid request body' }, { status: 400 })
      }

      // 3. Update inspection_notes — RLS enforces ownership via user_id guard
      const { error } = await supabase
        .from('assets')
        .update({ inspection_notes: notes })
        .eq('id', assetId)
        .eq('user_id', user.id)

      if (error) return Response.json({ error: error.message }, { status: 500 })
      return Response.json({ success: true })
    }
    ```
