**FORKLIFT SCHEMA — forklift.ts:**

Fields to flip to aiExtractable: true AND add aiHint:
- engine_model (sfOrder 2): `aiHint: 'Data plate or infer from make/model (e.g. Linde E50→electric motor, Toyota 7FBE20→electric, Komatsu FB20→Komatsu).'`
- max_lift_capacity (sfOrder 6): `aiHint: 'Forklift data plate: rated load capacity. Format: number with unit (e.g. 2500kg, 5T). Also inferrable from model number suffix.'`
- max_lift_height (sfOrder 7): `aiHint: 'Forklift data plate or mast specs: maximum lift height. Format: number with unit (e.g. 4500mm, 4.5m).'`
- colour (sfOrder 15): `aiHint: 'Exterior colour from photos. Common forklift colours: Yellow, Orange, Red, Blue, White, Green. Use brand colour as default if visible.'`
- transmission (sfOrder 17): `aiHint: 'Infer from make/model and fuel type. Electric: Electric Drive. LPG/Diesel: Automatic Torque Converter or Hydrostatic.'`
- fuel_type (sfOrder 19): `aiHint: 'Data plate or battery/engine type. Common: Electric, LPG, Diesel, Petrol. Electric forklifts have visible battery compartment.'`

Fields already aiExtractable: true — add aiHint:
- engine_manufacturer (sfOrder 1): `aiHint: 'Badge or infer from make/model (e.g. Toyota→Toyota, Linde→Linde electric, Komatsu→Komatsu, Crown→Crown electric).'`
- vin (sfOrder 4): `aiHint: 'Data plate or chassis: VIN if present. Never infer — only extract if directly visible.'`
- serial (sfOrder 5): `aiHint: 'Data plate: Serial Number. Typically alphanumeric. Never infer — only extract if directly visible.'`
- make (sfOrder 8): `aiHint: 'Data plate or cab badge: manufacturer name (e.g. Toyota, Linde, Komatsu, Crown, Hyster, Yale, Jungheinrich).'`
- model (sfOrder 10): `aiHint: 'Data plate: model designation (e.g. 7FBE20, E50, FB20, C5000, H50FT). Read exactly as printed.'`
- year (sfOrder 12): `aiHint: 'Data plate: Year of Manufacture or DOM. 4-digit year only.'`

Fields to keep aiExtractable: false: engine_number, cabin, attachments, tilt_degrees, truck_weight, tyres, stages, type, mast_type, extras, hours
