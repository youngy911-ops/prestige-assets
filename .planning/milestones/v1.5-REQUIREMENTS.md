## v1.5 Requirements

### Brand & Config

- [x] **BRAND-01**: QR code on output/report pages uses configurable domain (not hardcoded assetbookintool.com) — ✅ Phase 20, validated

### Accessibility

- [x] **A11Y-01**: BottomNav active link has aria-current="page" — ✅ Phase 23, all 3 nav links updated
- [x] **A11Y-02**: Expandable/collapsible sections (extraction "not found" fields) have aria-expanded attribute — ✅ Phase 23, ExtractionResultPanel updated
